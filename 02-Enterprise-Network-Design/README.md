# 🌐 Enterprise Network Design and Implementation for LankaTrade Holdings (Pvt) Ltd

## 📌 Project Overview

This project demonstrates the design and implementation of a secure enterprise network for **LankaTrade Holdings (Pvt) Ltd** using Cisco Packet Tracer. The network was designed using a hierarchical architecture to provide secure, scalable, and reliable connectivity between departments while supporting centralized services and secure remote administration.

---

# 🎯 Project Objectives

- Design a scalable enterprise network topology.
- Configure VLANs for departmental network segmentation.
- Implement inter-VLAN routing using multilayer switches.
- Configure a dedicated DHCP server for automatic IP address allocation.
- Implement OSPF dynamic routing between routers and multilayer switches.
- Configure SSH for secure remote device management.
- Implement Switch Port Security within the Finance and Accounts Department.
- Configure NAT Overload (PAT) for Internet connectivity.
- Apply Access Control Lists (ACLs) to control network traffic.
- Verify end-to-end network connectivity and functionality.
