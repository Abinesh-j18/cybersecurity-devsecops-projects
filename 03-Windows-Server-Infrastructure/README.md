# 🖥️ Windows Server Infrastructure & Active Directory Lab

## 📌 Project Overview

This project demonstrates the deployment of a virtual Windows Server 2012 infrastructure using VMware Workstation. The lab simulates an enterprise environment by configuring core Windows Server services including Active Directory, DHCP, File & Print Services, and Windows Firewall.

---

# 🎯 Objectives

- Build a virtual enterprise network using VMware
- Configure Windows Server 2012 roles
- Implement centralized user authentication
- Configure automated IP address allocation
- Deploy file and printer sharing services
- Practice Windows Server administration and troubleshooting

---

# 🛠️ Technologies Used

- VMware Workstation
- Windows Server 2012
- Active Directory Domain Services (AD DS)
- DHCP
- File & Print Services
- Windows Firewall
